# SDL_PongGame
2人プレイと複数ボールに対応したPongゲームです

![](https://github.com/Daiki-Iijima/SDL_PongGame/blob/main/demo.gif)

# SDL2のインストール
homebrewでインストールして開発しています。
```bash
brew install SDL2
```

# コンパイル手順
makeでビルドしてください
```
make
```

# ユーザー数の設定方法
実行時の引数にユーザー数を入力してください

- 1人の場合
```bash
./main 1
```
- 2人の場合
```bash
./main 2
```
